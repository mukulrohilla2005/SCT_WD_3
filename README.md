# Tic-Tac-Toe Web Application 🎮

A responsive, interactive Tic-Tac-Toe game built with vanilla HTML, CSS, and JavaScript

## Features
- 👥 Player vs Player mode
- 🤖 Player vs Computer mode
- 🧠 Three AI difficulty levels — Easy, Medium, Hard (Minimax with Alpha-Beta Pruning)
- 🏆 Live score tracking for X, O, and Draws
- ⚡ Win detection with cell highlighting
- 🔄 New Game and Reset Scores controls
- 📱 Fully responsive — works on mobile and desktop

## Tech Stack
- HTML5
- CSS3 (CSS Variables, Animations, Flexbox)
- JavaScript (ES6+, Minimax Algorithm)

## AI Logic
The computer opponent uses the **Minimax algorithm with Alpha-Beta Pruning** for the Hard difficulty, making it unbeatable. Medium difficulty uses a 50/50 mix of random and optimal moves. Easy difficulty plays randomly.
